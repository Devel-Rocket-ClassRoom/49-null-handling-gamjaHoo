# 코딩 연습 - 널(null) 다루기 (2)

`Program.cs`에 아래 과제들의 코드를 순서대로 작성하세요.
모든 과제를 작성한 후 실행하면 각 과제의 실행 결과가 순서대로 출력됩니다.

> 💡 변수 이름이 겹치는 경우 중괄호 `{ }`로 코드를 감싸면 각 블록에서 같은 이름을 사용할 수 있습니다.

## 1. 널 병합 할당 연산자 (??=)

### 1-1. 기본 사용법

문자열 변수 `name`을 `null`로 선언하고, `??=` 연산자로 `"이름 없음"`을 할당한 후 출력하세요. 다시 `??=`로 `"새 이름"`을 할당 시도하고 출력하세요.

**실행 결과**

```
이름 없음
이름 없음
```

### 1-2. if문과의 비교

문자열 변수 `playerName`을 `null`로 선언하고, if문으로 null이면 `"플레이어"`를 할당하여 출력하세요.

**실행 결과**

```
플레이어
```

### 1-3. 지연 초기화

`List<string>` 변수 `_items`를 `null`로 선언하고, `??=`로 새 리스트를 생성한 뒤 `"검"`을 추가하세요. 다시 `??=`로 리스트 생성을 시도하고(이미 있으므로 무시됨) `"방패"`를 추가하세요. 아이템 수와 각 아이템을 출력하세요.

**실행 결과**

```
아이템 수: 2
- 검
- 방패
```

## 2. 널 조건 연산자 (?.)

### 2-1. 기본 사용법

문자열 변수 `message`를 `null`로 선언하고, `?.Length`로 길이를 `int?` 변수에 저장하여 출력하세요. 그 다음 `message`에 `"안녕하세요"`를 할당하고 같은 방식으로 출력하세요.

**실행 결과**

```
길이:
길이: 5
```

### 2-2. NullReferenceException 방지

문자열 변수 `text`를 `null`로 선언하고, `?.Length`로 안전하게 길이를 가져온 뒤, 결과가 `null`이면 "텍스트가 없습니다."를 출력하세요.

**실행 결과**

```
텍스트가 없습니다.
```

### 2-3. 메서드 호출에 사용

문자열 변수 `name`을 `null`로 선언하고, `?.ToUpper()`의 결과를 출력하세요. 그 다음 `name`에 `"hello"`를 할당하고 같은 방식으로 출력하세요.

**실행 결과**

```
대문자:
대문자: HELLO
```

### 2-4. 체이닝

문자열 변수 `text`를 `null`로 선언하고, `?.ToUpper()?.Trim()`의 결과를 출력하세요. 그 다음 `text`에 `"  hello  "`를 할당하고 같은 방식으로 출력하세요.

**실행 결과**

```
결과:
결과: HELLO
```

### 2-5. 컬렉션과 함께 사용

`List<string>` 변수 `items`를 `null`로 선언하고, `?.Count`를 `int?` 변수에 저장하여 출력하세요. 그 다음 `items`에 `"사과"`, `"바나나"`를 담은 리스트를 할당하고 같은 방식으로 출력하세요.

**실행 결과**

```
아이템 수:
아이템 수: 2
```

### 2-6. 인덱서와 함께 사용 (?[])

문자열 변수 `text`를 `null`로 선언하고, `?[0]`으로 첫 글자를 `char?` 변수에 저장하여 출력하세요. 그 다음 `text`에 `"Hello"`를 할당하고 같은 방식으로 출력하세요.

**실행 결과**

```
첫 글자:
첫 글자: H
```

## 3. 연산자 조합

### 3-1. ?.와 ??의 조합

문자열 변수 `message`를 `null`로 선언하고, `?.Length ?? 0`으로 길이를 `int` 변수에 저장하여 출력하세요. 그 다음 `message`에 `"Hello"`를 할당하고 같은 방식으로 출력하세요.

**실행 결과**

```
길이: 0
길이: 5
```

### 3-2. 컬렉션에서의 조합 활용

`List<string>` 변수 `names`를 `null`로 선언하고, `?.Count ?? 0`으로 아이템 수를 출력하세요. 그 다음 `names`에 `"Kim"`, `"Lee"`, `"Park"`을 담은 리스트를 할당하고 같은 방식으로 출력하세요.

**실행 결과**

```
이름 수: 0
이름 수: 3
```

### 3-3. 실용적인 예제

문자열 변수 `playerName`을 `null`로, `int?` 변수 `playerLevel`을 `null`로, `List<string>` 변수 `inventory`를 `null`로 선언하세요. `??`와 `?.`를 조합하여 `displayName`에 `"Unknown"`, `level`에 1, `itemCount`에 `inventory?.Count ?? 0`을 할당하고 출력하세요.

**실행 결과**

```
플레이어: Unknown
레벨: 1
인벤토리 아이템: 0개
```
