# NullableScoreBoard

5개 경기의 점수를 관리하는 프로그램을 작성하세요.

아직 진행되지 않은 경기는 점수가 null이고, 진행된 경기만 실제 점수를 가집니다. Nullable 형식의 배열을 사용하여 경기 상태를 구분하고 통계를 계산하세요.

## 요구사항

1. `int?[]` 배열로 5개 경기 점수를 선언하세요:
   - `int?[] scores = { 3, null, 7, null, 5 };`

2. 배열을 순회하면서 각 경기 상태를 출력하세요:
   - `HasValue`로 진행 여부를 확인
   - 진행된 경기: "경기 N: O점 (진행 완료)"
   - 미진행 경기: "경기 N: 미진행"

3. `GetValueOrDefault(-1)`을 사용하여 모든 경기 점수를 출력하세요:
   - 미진행 경기는 -1로 표시

4. 진행된 경기만 합산하여 총점과 평균을 계산하세요:
   - 평균은 소수점 첫째 자리까지 출력

## 예상 실행 결과

```
=== 경기 상태 ===
경기 1: 3점 (진행 완료)
경기 2: 미진행
경기 3: 7점 (진행 완료)
경기 4: 미진행
경기 5: 5점 (진행 완료)

=== 전체 점수 (미진행 = -1) ===
경기 1: 3
경기 2: -1
경기 3: 7
경기 4: -1
경기 5: 5

=== 통계 ===
진행된 경기 수: 3
총점: 15
평균: 5.0
```
