# SafeInventory

여러 플레이어의 인벤토리를 안전하게 조회하는 프로그램을 작성하세요.

일부 플레이어는 아직 인벤토리가 생성되지 않았거나(null), 일부는 빈 인벤토리를 가지고 있습니다. 널 조건 연산자(`?.`)와 널 병합 연산자(`??`)를 조합하여 안전하게 인벤토리 정보를 출력하세요.

### 요구사항

1. 세 명의 플레이어 인벤토리를 선언하세요:
   - `player1Items`: null (인벤토리 미생성)
   - `player2Items`: 빈 리스트 (인벤토리 있지만 비어있음)
   - `player3Items`: "검", "방패", "포션" 세 개의 아이템

2. 각 플레이어에 대해 다음을 출력하세요:
   - 인벤토리 아이템 수: `?.Count ?? 0` 사용
   - 첫 번째 아이템: `?[0] ?? "없음"` 사용
   - 마지막 아이템: `?[^1] ?? "없음"` 사용 (인덱스 연산자 사용)

3. NullReferenceException 없이 안전하게 처리해야 합니다.

## 예상 실행 결과

```
=== 플레이어 1 (인벤토리 없음) ===
아이템 수: 0
첫 번째 아이템: 없음
마지막 아이템: 없음

=== 플레이어 2 (빈 인벤토리) ===
아이템 수: 0
첫 번째 아이템: 없음
마지막 아이템: 없음

=== 플레이어 3 (아이템 보유) ===
아이템 수: 3
첫 번째 아이템: 검
마지막 아이템: 포션
```
